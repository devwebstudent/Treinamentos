Adicionando Fontes com CSS

Propriedades das fontes

font-style: italic; = adiciona itálico.

font-weight: bold; = adiciona negrito.

font-size: 1em; = para mudarmos o tamanho da fonte.

font-family: Arial, sans-serif; = para mudarmos a fonte da nossa página.

font-variant: small-caps; = as letras minúsculas serão convertidas para letras maiúsculas.

line-height: 1em; = para aumentarmos o espaçamento de cada linha do nosso texto.

Shorthand da propriedade font

Para o shorthand da propriedade font, vamos utilizar a seguinte ordem:

  1. font-style
  2. font-weight
  3. font-size
  4. font-family

Nosso shorthand ficará como no exemplo abaixo: 

        
            font: italic bold 1.5em Arial, sans-serif;

Alinhamentos

text-align: = para alinharmos nosso texto. Valores:

  • left
  • right
  • center
  • justify

text-indent: 30px = adiciona recuo ao nosso texto, como em um determinado parágrafo, por exemplo.
