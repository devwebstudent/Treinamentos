HTML Semântico

Títulos: h1... h6

Parágrafos: p

Negrito: tag <strong>

Itálico: tag <em>

Texto Inserido (sublinhado): tag <ins>

Texto deletado (riscado): tag del

Marca Texto: tag mark

Texto Sobrescrito: tag sup

Texto Subscrito: tag sub

texto pequeno: tag small

Para inserior código de programação: tag code

Pré-formatação (para ficar igual como está no VS Code: tag pre

Abreviação (passe o mouse AQUI): tag abbr

Citação: Há momentos em que a maior sabedoria é parecer não saber nada: tag q

Citação longa: tag blockquote
